# scrap-yard
A getting started with Typescript project

### Environment Setup:

#### Installation

1. setup Typescript.

```
npm install -g typescript ts-node
```

#### verify

```
tsc --help
```

### First App

Generate package.json file:

```
npm init -y
```

Install Axios

```
npm install axios
```

#### Compile & Run Typescript code:

compile:

```
tsc index.ts
```

run:

```
node index.js
```

compile & run:

```
ts-node index.ts
```
